Controls - 

	Left/Right - Horizontal
  
	Forward/Back - Vertical
  
	Look Horizontal/Vertical - (Mouse Position)
  
	Interact (Space)
 
	Throw (Mouse Click)
  
	Reload (Right Mouse Click)
  
	Pause (P or ESC)
